# 土狗音乐播放器

> 一个基于 electron-vue 开发的音乐播放器


##### 运行

``` bash
git clone https://github.com/SmallRuralDog/electron-vue-music.git

cd electron-vue-music

npm install

# 运行开发模式
npm run dev

# 打包安装文件 
npm run build

```
>注意 正式打包需要把图标样式下载到本地，否则正式版图标不显示
